Basic Internet Technologies and It's Impact in our Daily Life

The Internet is a connection of networks that uses the OSI Model and TCP/IP Model to transfer files or data from one end of devices to another. There are 7 layers on OSI Model and 4 layers on TCP/IP layers which have a similar function. The Internet has made a positive and negative impact on our life by making our work easy to do, entertain us, and lazy too. Works that require a lot of time are made easy and instant by using internet technology. From the purchase of goods to payment of the goods have been made online due to Internet Technologies. Internet uses a different set of protocols to complete the given tasks such as HTTP, FTP, and telnet.

Internet is a group of networks that consists of many local Area Networks that are connected. Internet is run by tens of thousands of interconnected networks which are run by service providers, individual companies, universities, governments, and others (Internet Society, 2018). OSI and TCP/IP model is used to transform data and information from one computer to another connected on the Internet. TCP/IP protocol uses an entire suite of protocols, in which Internet Protocol(IP) and Transport Control Protocol (TCP) are most important. (WEBSYDIAN, 2021)

There are 7 Layers in the OSI Model:

  • Application Layer :
    • User interacts with this layer for web browsing and internet-connected application use application layer.
  • Presentation Layer :
    • The presentation layer is used for translating data from application formatting to network formatting and vice versa.
  • Session Layer :
    • The session layer is used for establishing and terminating the connection between devices.
  • Transport Layer :
    • Transport Layer is one of the important layers that transfer data between system and host which is capable of error-checking and data recovery.
  • Network Layer :
    • Network Layer determines the process of sending data to receiving devices by packet forwarding, routing, and addressing.
  • Data Link Layer :
    • Data link layer covers binary into signals and also allows different layers to access media.
  • Physical Layer :
    • Physical Layer is the actual hardware that transmits signals over medium.

TCP/IP Model uses 4 different layers which are:

  • Application Layer :
    • The application layer is called the Process layer which combines of Session layer, Presentation layer, and Application layer of the OSI Model.
  • Transport Layer :
    • The transport layer is called the host-to-host layer which has a similar function as the Transport Layer of the OSI Model.
  • Internet Layer :
    • The transport layer is called the host-to-host layer which has a similar function as the Transport Layer of the OSI Model.
  • Network Layer :
    • Network is called Network Interface layer which has a similar function as Data Link layer and Physical Layer.
(Alienor, 2018)

Working Mechanism of Network Layer:

Suppose two users are using Skype so skype is a network-connected application that uses the application layer to send you videos or photos and messages using different protocols such as Telnet and File Transfer Protocol (FTP).
The presentation layer is responsible for translating received data into binary and compressing it with encryption. When the user gets your data, it is also responsible for decrypting it.
When text files or image files are being downloaded then the Session layer is used in the arrangement of packets and establishes, maintains, and ends communication between devices.
Transport Layer segments received data and ensure that the segment reaches the correct application by assigning a source, destination port number, and sequence number as well. The transport layer is also responsible for controlling the amount of data transmitted that can be handled by the receiver end devices and ensures that data is successfully transferred by checking for errors also.
The network layer transmits each segment between devices in form of packets. This layer assigns source and destination IP addresses to each segment and determines the best paths for data transfer.
Data Link layer adds sender and receiver MAC addresses to the data packet to form a frame and enables frames to be transported using Network Interface Card (NIC). This layer allows the network layer to access media and controls the placement of data and receive of data from media.
The physical layer converts binary into signals and transmits them over local media depending upon the medium. They can be electrical, light, radio signals, etc.
(Alienor, 2018)

Some of the common Internet Protocols are TCP/IP protocol, HTTP (HyperText Transfer Protocol), and FTP (File Transfer Protocol).

  • TCP/IP:
    • TCP/IP protocol is that guarantees the arrival of any data transmission between two endpoints. It is also called peer-to-peer protocol as only two endpoints are connected.
  • HTTP:
    • HTTP is a protocol that is used to transmit data present on World Wide Web (WWW) which includes text, multimedia, and graphics.
  • FTP:
    • FTP is a protocol that is used during the transmission of files between computers connected by a TCP/IP network.
(Wikibooks, 2021)

Impact of Internet on Daily Life

Internet may seem something very simple, but it is one such powerful weapon with which one can do almost every activity. Nowadays, there is use the Internet for almost everything from communication to shopping, from reading news to ordering food, from education to booking tickets for movies. Internet is vastly used in every small as well as huge sectors. It has been an important part of our life. Whether it's personal life or professional life, the Internet is used widely.
We can take an example of the current situation. How are humans able to live in this pandemic? A simple answer- Internet. All the online classes, work from home, entertainment like movies, games are only possible because of the Internet.
As we all know everything comes with its pros and cons. But the Internet if used correctly can overcome the disadvantages.

Positive Impact of Internet

  1. Internet is a great platform for effective and fast communication through email and instant messaging service to any corner of the world.
  2. Internet banking has made human life less complicated. Banking transactions can be done staying at home. There is so need to go miles just to pay electricity bills, as that can be done staying home using the Internet.
  3. Nowadays, knowledge is not just limited to books, journals, and libraries. The Internet has made the education process effective. Students should not collect huge numbers of books and magazines for research, Internet has everything. Students can also attend online classes through the Internet.
  4. Online services like online shopping, ordering food have been possible due to the Internet. Just a click or two we get our order in front of our doors.
  5. Applications for jobs have also been easier. Most of the job vacancies are advertised online with online forms. Nowadays, even the interview can be done online.

Negative Impact of Internet

  1. Internet has caused a lot of health problems. Due to the internet, people use their devices for a long period which causes eye problems, headache, backache, etc.
  2. People misuse the Internet for hacking, pornography, spreading hate, cyberbully, etc.
  3. Internet can be addictive without us realizing it. This is a common problem among young adults and teenagers which leads to lack of sleep.
